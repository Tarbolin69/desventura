# Desventura

Un programa para la creacion de aventuras de texto usando archivos csv.

Si se sigue el formato creado por desventura-desarollo, todo juego tendria que poder jugarse. Modificaciones al archivo resultante pueden causar que el juego se rompa si no se respeta dicho formato.

En cada juego ha de haber 1 item oculto que sera necesario para ganar el juego (dicho item empieza con *). Debes de usar este item en una ubicacion especifica en el mapa para ganar el juego.
